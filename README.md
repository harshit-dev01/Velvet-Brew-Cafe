# ☕ Velvet Brew Cafe Website

A modern, responsive cafe website designed to showcase a premium coffee experience with stylish UI, smooth animations, and interactive elements.


## 📌 Project Overview

Velvet Brew Cafe is a visually rich front-end web project that represents a contemporary urban cafe. The design focuses on aesthetics, user experience, and smooth navigation.

This website includes:

* Hero section with modern layout
* Interactive menu cards with hover effects
* Gallery section with visual storytelling
* Responsive design for all devices
* Stylish footer with contact and newsletter

## 🛠️ Technologies Used

* HTML5
* CSS3 (Custom styling, animations, gradients)
* Google Fonts

## 🎨 Features

* Fully responsive layout
* Smooth animations and transitions
* Modern UI/UX design
* Hover effects on cards and images
* Organized sections (About, Menu, Gallery, Contact)
* Clean and structured code
